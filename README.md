 # 💼 Job Portal Application

A modern **Job Portal Web Application** built with **Next.js** where **Admins can post jobs** and **Users can apply for jobs**.  
The platform allows efficient job management and a smooth application process for candidates.

---

## 🚀 Features

### 👤 User Features
- Create an account and login
- Browse available jobs
- View job details
- Apply for jobs
- Track applied jobs

### 🛠️ Admin Features
- Secure admin login
- Create and publish job listings
- Manage posted jobs
- View applicants for each job

---

## 🧰 Tech Stack

- **Frontend:** Next.js (App Router)
- **Backend:** Next.js API Routes
- **Database:** MongoDB
- **Authentication:** JWT / Session-based
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
